Formation ChatGPT

Bienvenue dans cette formation interactive dédiée à l'utilisation de ChatGPT.

1. Comprendre ChatGPT

ChatGPT est un assistant conversationnel capable de générer du texte en fonction de vos instructions. Plus votre demande est précise, plus la réponse sera pertinente.

2. Rédiger des prompts efficaces

Formulez toujours votre demande avec un contexte clair et un objectif précis pour obtenir une réponse pertinente.

3. Exemples pour la rédaction

Demandez la rédaction d'emails ou de textes courts en détaillant le ton souhaité.

4. Exemples pour l'analyse

Fournissez un texte à résumer ou à analyser pour obtenir les points clés ou un style d'écriture.

5. Exemples pour le code

  • Écris une fonction JavaScript qui trie un tableau
  • Montre-moi un exemple d'appel API en Python

6. Bonnes pratiques

Structurez vos questions, précisez le format attendu et n'hésitez pas à poser des questions de suivi.

7. Personnalisation des réponses

Indiquez le public ciblé ou le niveau de langage souhaité pour adapter les réponses.

8. Résumé

En combinant contexte, exemples et questions de suivi, vous optimisez la qualité des réponses de ChatGPT.

9. Quiz final

1. Quelle est la meilleure façon de formuler un prompt ?

2. Comment demander un exemple de code ?

3. Existe-t-il un raccourci pour régénérer une réponse ?
