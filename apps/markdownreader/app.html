Aide Markdown

Syntaxe de base :

  • # Titre 1 → Titre principal
  • ## Titre 2 → Sous-titre
  • **gras** → gras
  • *italique* → italique
  • [lien](url) → lien
  • `code` → code
  • - item → Liste à puces
  • 1. item → Liste numérotée
  • > citation → Citation
